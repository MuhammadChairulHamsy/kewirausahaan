## ğŸš€ How to Run

1. **Navigasi ke direktori proyek**

   ```bash
   git clone https://github.com/MuhammadChairulHamsy/kewirausahaan
   ```

2. **Navigate to project directory**

   ```bash
   cd kewirausahaan
   ```

3. **Buka terminal di folder utama proyek, lalu jalankan perintah:**

   ```bash
   npm install
   ```

4. **Menjalankan Tailwind CSS (Watch Mode)**
   **_Agar setiap perubahan class Tailwind langsung teraplikasi ke desain, jalankan perintah berikut di terminal:_**

   ```bash
   npm run watch
   ```

   **Catatan: Jangan tutup terminal ini selama Anda melakukan perubahan kode agar file output.css terus diperbarui secara otomatis.**

5. **Menjalankan di Browser**
   1. Install Extension vscode "live server"
   2. Klik kanan pada file index.html.
   3. Pilih "Open with Live Server" atau klik tombol "Go Live" di pojok kanan bawah VS Code.
   4. Browser akan terbuka otomatis di alamat http://127.0.0.1:5500.
